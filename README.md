---

## Istallation

### Install dependencies

```sh
npm i
```
or
```sh
yarn
```

### Install TypeScript as a Node.js package

```sh
npm i -g typescript
```

or

```sh
yarn global add typescript
```

---

## Running in dev mode

```sh
npx next dev
```
